# 💼 Personal Portfolio Website
> **Level 1 - Task 2**

## 📖 Overview

The **Personal Portfolio Website** is a responsive and modern web application .
This portfolio serves as a digital résumé, showcasing my technical skills, projects, educational background, and contact information. The website is designed to provide visitors with a professional overview of my work while demonstrating my front-end web development skills using **HTML5**, **CSS3**, and **JavaScript**.

---

## 🎯 Objective

The primary objective of this project is to create a professional portfolio website that:

- Introduces me as a web developer.
- Highlights my technical skills.
- Showcases my projects.
- Provides my contact information.
- Demonstrates responsive web design and modern UI principles.

---
